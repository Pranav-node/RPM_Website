# RPM Website

A modern web application for showcasing and managing luxury vehicles with an admin panel and Supabase backend integration.

## 🚀 Live Demo

https://rpm-umber.vercel.app

## 📷 Features

- Modern responsive UI
- Luxury car showcase
- Admin dashboard
- Image gallery
- Supabase database integration
- SQL setup scripts
- Vercel deployment ready

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Supabase
- Vercel Serverless Functions

## ⚙️ Running Locally

Clone the repository

```bash
git clone https://github.com/Pranav-node/RPM_Website.git
```

Open the project

```bash
cd RPM_Website
```

### Option 1

Open `index.html` directly in your browser.

### Option 2 (Recommended)

Run a local server

```bash
npx serve .
```

or

```bash
npx live-server
```

## 🚀 Deployment

Deploy instantly on Vercel.

```bash
vercel
```

## 🗄 Database

The repository includes SQL scripts for Supabase setup.

- `supabase-setup.sql`
- `fix-policies.sql`
- `fix-all-policies.sql`

Import these into your Supabase project before running the application.
